#  IPL Power BI Analysis (2008–2025)
 **If you find this project useful, consider giving it a star!**

##  Project Overview

This project presents an **interactive Power BI dashboard** analyzing Indian Premier League (IPL) data from **2008 to 2025**.  
It provides insights into teams, players, matches, runs, sixes, fours, centuries, venues, and overall tournament performance.

The dashboard is designed for **easy understanding, storytelling, and data-driven insights**.



##  Dashboard Highlights

- Total Seasons, Matches, Teams, and Venues
- Champion Teams Overview
- Orange Cap & Purple Cap Statistics
- Total Runs, Fours, Sixes, Centuries & Half-Centuries
- Team-wise Performance & Points Table
- Player Performance Analysis
- Season-wise Insights (2008–2025)



##  Tools & Technologies Used

- **Microsoft Power BI** – Dashboard creation & data visualization  
- **DAX (Data Analysis Expressions)** – Calculated measures & KPIs  
- **Power Query** – Data cleaning and transformation  
- **CSV Datasets** – IPL historical data  
- **GitHub** – Project hosting, version control, and documentation

##  Dataset
Due to GitHub file size limitations, the IPL datasets are provided as a compressed ZIP file.

**File:** `dataset.zip`

After downloading:
1. Extract the ZIP file
2. You will find the following CSV files:
   - `ball_by_ball_data.csv`
   - `ipl_matches_data.csv`
   - `players-data-updated.csv`
   - `teams_data.csv`


##  How to Use the Project

1. Download or clone the repository  
2. Extract the dataset ZIP (if applicable)  
3. Open `IPL_ANALYSIS_POWER_BI.pbix` using **Power BI Desktop**  
4. If required, update dataset paths in **Power Query**  
5. Refresh data to explore the dashboard
